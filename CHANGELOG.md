# 更新日志

## 2026-04-13

### 新增功能

#### 饼图编辑器（PieEditor）
- 扇形内同时显示数值和百分比
- 双击标题可编辑
- 双击图例可修改标签和数值
- 点击图例项显示删除按钮
- 支持拖拽图例项排序
- 添加项目按钮

#### 柱状图/折线图编辑器（XyChartEditor）
- 柱子顶部默认显示数值标签
- 拖拽柱子/折线点上下改变数值，实时显示 tooltip
- 双击 X 轴标签可编辑
- 双击图例可重命名系列
- 列操作栏（始终显示，未选中时禁用）：左插列、右插列、删除列、拖移换位
- `+ 列` 按钮快速追加列

#### 时序图编辑器（SequenceEditor）
- 全新 SVG 自定义渲染，替代错误的 Dagre 布局
- 参与者横向排列，生命线垂直，消息水平
- 双击参与者名称可编辑
- 双击消息标签可编辑
- 点击消息选中，显示操作栏：上移/下移/切换线型/修改发送方和接收方/删除
- `+ 消息`、`+ 参与者`、删除参与者按钮

### 交互改进

#### 手动同步机制
- 饼图、柱状图、时序图不再自动同步
- 代码区"同步至画布"：将代码解析到可视化编辑器
- 编辑区"同步回代码"：将可视化编辑内容写回代码

### Bug 修复

- 修复 `D-->>B` 时序图语法解析错误（正则顺序问题，`-->>` 被 `-->` 提前匹配）
- 修复时序图序列化缩进改为 4 空格，与 Mermaid 标准一致
- 修复柱状图插列/删列操作中 `xLabels` 长度不一致导致的 splice 错误
- 修复删列后 `selectedCol` 未清除导致状态残留
