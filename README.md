# Mermaid 可视化编辑器

将图片通过 AI 转换为 Mermaid 图表代码，并提供可视化编辑、实时预览的一体化工具。

## 功能

- **图片导入**：上传 PNG/JPEG 图片，AI (Claude) 自动识别并生成 Mermaid 代码
- **模板库**：内置多种模板（通用示例 + 车载 AUTOSAR 场景）
- **收藏夹**：保存常用图表，支持加载、重命名、删除，数据持久化到 localStorage
- **代码编辑**：实时编辑 Mermaid 代码
- **可视化编辑**：基于 Dagre 布局的交互式画布编辑
  - 拖拽移动节点和子图（子图拖拽时内部节点跟随移动）
  - 点击连线、选中边编辑样式
  - 右侧面板修改节点标签(Label)和唯一标识(ID)
  - 节点颜色和边样式的可视化配置
  - 平行边（同一对节点间的多条边）智能分离显示
  - 子图支持：框选创建、拖拽放置、全区域拖动、节点自动归属
  - 智能边路径：16 种连接点组合，自动选择最短路径，支持贝塞尔曲线
  - 格式化布局：一键重新运行 Dagre 布局算法，整理节点和子图位置
- **画布操作**：鼠标滚轮缩放、中键/右键拖拽平移、框选创建节点/子图
- **图表设置**：布局方向（切换时自动重排布局）、主题、外观、曲线样式、背景网格
- **专用编辑器**：
  - 饼图、XY 图表各有独立的可视化编辑界面
  - 时序图编辑器：
    - 右侧面板提供参与者和片段基本元素，支持拖拽/点击放置
    - 参与者支持拖拽排序、双击编辑、右键菜单
    - 消息连线：点击生命线发起连线，从点击位置出发
    - 片段支持：loop/alt/opt/par/critical/break，点击后框选消息行范围创建
    - 激活条：支持 activate/deactivate 显式语法，右侧面板可为参与者添加/删除激活条
    - 选中元素后右侧面板显示属性编辑（ID、标签、类型、样式）
    - 画布上显示参与者和片段的唯一 ID
    - 所有元素（参与者、消息、片段）均可右键引用到 AI 助手
  - 思维导图编辑器：
    - 左侧树形视图：折叠/展开、双击编辑标签、添加/删除节点
    - 中间画布预览：贝塞尔曲线连线，点击节点选中
    - 右侧属性面板：修改标签、形状（默认/方形/圆角/圆形/感叹/云形）、添加子节点/兄弟节点
  - 树形图编辑器：节点树支持添加子节点/兄弟节点/根节点、删除、上移/下移、缩进/提升，右侧属性面板编辑标签
  - 看板编辑器：支持添加/删除列（作用于画布），左键平移画布，卡片属性含负责人（assignee）和优先级（priority）
  - 数据包图编辑器：字段支持左右 resize 控制条（选中时显示），拖拽移动，push/覆盖碰撞模式
- **渲染预览**：Mermaid 原生渲染，自动居中适配显示，支持导出 PNG 和 SVG 格式（高清晰度）
- **AI 助手**：内置 AI 对话框，支持自定义 API 配置，可引用节点/子图/时序图元素上下文，智能生成或修改图表

**三栏说明**：

## 快速开始

### 环境要求

- Node.js 16.x 或更高版本
- npm 包管理器

### 安装与运行

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量（图片转 Mermaid 功能需要）
cp .env.example .env.local
# 编辑 .env.local，填入 Anthropic API Key：
# ANTHROPIC_API_KEY=sk-ant-xxx
# ANTHROPIC_BASE_URL=https://api.anthropic.com  (可选，自定义 API 地址)

# 3. 启动开发服务器
npm run dev
# 访问 http://localhost:3333
```

### Windows 一键启动

双击 `start.bat`，脚本会自动：

1. 检查 Node.js 是否已安装
2. 首次运行时自动安装依赖
3. 打开浏览器访问 `http://localhost:3333`
4. 启动 Next.js 开发服务器

### 构建生产版本

```bash
npm run build
npm start
```

## 架构设计

本项目采用 **解耦的双向同步架构**，将原生的 Mermaid 渲染与自定义的交互式画布分离：

1. **渲染区**：完全交由 Mermaid.js 原生接管，保证渲染效果的绝对正确性。
2. **画布区**：完全交由自定义的 SVG 画布和 Dagre 算法接管，提供丰富的交互能力（拖拽、连线、修改属性等）。
3. **双向同步机制**：
   - **读取代码**：清空当前画布临时状态，读取源码区代码，通过内部 Parser 解析出逻辑拓扑（节点、边、子图），并用 Dagre 进行初始坐标排版，最终在画布区重建交互视图。
   - **回写代码**：将画布区当前的拓扑关系、节点属性和全局配置，通过内部 Serializer 重新序列化为标准的 Mermaid 代码并覆盖源码区。

## 使用指南

### 基本工作流

1. **编辑源码**：在代码编辑器中输入 Mermaid 代码，或从左侧选择预设模板。
2. **预览与渲染**：中间的"渲染区"会实时预览原生渲染效果。
3. **进入可视化编辑**：点击画布区的"⬇️ 读取代码"按钮，代码将被解析并在画布上生成可拖拽的拓扑图。
4. **可视化操作**：
   - **添加节点**：从左侧面板拖拽形状到画布，或点击形状按钮后在画布上点击/框选放置
   - **添加子图**：点击"添加子图"按钮后在画布上框选区域，或拖拽到画布
   - **拖拽**：按住节点或子图任意空白区域移动，节点拖入子图自动归属
   - **连线**：点击源节点右侧的连接手柄，拖动至目标节点释放
   - **修改属性**：选中节点/边后，在右侧面板修改 ID、标签、形状、颜色、样式
   - **调整布局**：在右侧"流程图设置"中修改布局方向和曲线样式（实时预览画布）
5. **保存与同步**：编辑完成后，点击"⬆️ 回写代码"按钮，所有修改（包括布局设置）会生成标准 Mermaid 代码并更新到代码区。

### 导出图表
- 在**渲染区**点击"导出 PNG"或"导出 SVG"按钮。
- 本项目采用了安全的内联 XMLSerializer 导出技术，生成的 PNG 图片默认为 2x 高清分辨率。

### AI 助手

1. **配置 AI**：在左侧面板底部填入 Base URL、模型名称和 API Key（支持 Anthropic 及兼容 OpenAI 格式的接口）。
2. **打开对话框**：画布区底部常驻 AI 助手栏，点击展开。
3. **引用节点/子图**：右键节点或子图，选择"引用到 AI"，节点名称会以标签形式插入输入框。
4. **提问**：在输入框中组织语句，如"在「节点A」右边增加一个矩形节点"，按 Enter 发送。
5. **应用结果**：AI 返回的 Mermaid 代码会自动应用到画布预览；确认无误后点击"回写代码"同步到编辑器。
6. **清空对话**：点击对话框右上角的清空按钮重置历史。

## 常见问题

**Q: 为什么改了节点文字或布局方向，代码区没有马上变化？**
A: 画布区和代码区是**完全解耦**的。这样做是为了防止频繁解析导致的性能问题和状态丢失。修改完成后，请点击"⬆️ 回写代码"按钮主动同步。布局方向和曲线样式会实时更新画布预览，但只在回写代码时才应用到代码区和渲染区。

**Q: 为什么渲染区是空的或者报错了？**
A: 当代码编辑区内的语法存在错误时，渲染区可能会显示解析失败的提示。画布区也有自己独立的空状态提示。

**Q: 图片转换 API 报错？**
A: 请检查 `.env.local` 中的 `ANTHROPIC_API_KEY` 是否正确配置。如果使用代理，需设置 `ANTHROPIC_BASE_URL`。
